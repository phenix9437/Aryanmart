# Badge

## Purpose
Generic status and label badges — savings, new, sale, role tags.

## Variants
| Variant | Example |
|---------|---------|
| `default` | Neutral label |
| `accent` | "Save 21%" |
| `success` | "Verified Purchase" |
| `govt` | Govt procurement tags |

## Props
| Prop | Type | Description |
|------|------|-------------|
| `variant` | string | Visual variant |
| `children` | ReactNode | Badge text |
