# Footer

## Purpose
Site-wide footer with links, contact, and trust badges.

## Layout
4-column (desktop): Logo + social + GSTIN/CIN | Quick Links | Support | Office Address  
Bottom bar: Copyright, legal links, ISO + Make in India badges

## Props
| Prop | Type | Description |
|------|------|-------------|
| `gstin` | string | Company GSTIN display |
| `cin` | string | Company CIN |

## Responsive
- Tablet: 2 columns
- Mobile: accordion sections
